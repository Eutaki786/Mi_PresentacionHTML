# Portafolio Profesional - Tomas Cardenas

Este proyecto es un portafolio personal desarrollado para el módulo de **Programacion Front-End** en **INACAP Concepción-Talcahuano**. El sitio presenta mi perfil como estudiante de Ingeniería en Ciberseguridad, mis habilidades técnicas y proyectos destacados.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica completa.
* **CSS3:** Diseño personalizado, Flexbox, CSS Grid y Media Queries.
* **Google Fonts:** Integración de la fuente "Carrois Gothic SC".
* **Git/GitHub:** Control de versiones con metodología de commits progresivos.

## 📱 Características
* **Diseño Responsivo:** Adaptable a dispositivos móviles, tablets y escritorio.
* **Semántica Web:** Uso correcto de etiquetas `<header>`, `<main>`, `<section>`, `<article>`, `<aside>` y `<footer>`.
* **Interactividad:** Formulario de contacto funcional (validación HTML5) y enlaces a repositorios externos.
